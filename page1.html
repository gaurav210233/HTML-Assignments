Understanding HTML Structure

An HTML document consists of several elements that work together to create a web page. Let's break down the basic structure:

  • <!DOCTYPE html>: This declaration defines the document type and version of HTML being used. In this case, it's HTML5.
  • <html>: The <html> element is the root of the HTML document and contains all other elements.
  • <head>: The <head> section contains metadata about the document, such as the title, character encoding, and links to external resources.
  • <title>: Inside the <head>, the <title> element sets the title of the web page, which appears in the browser's title bar or tab.
  • </head>: This tag closes the <head> section.
  • <body>: The <body> element contains the visible content of the web page, including text, images, links, and more.
  • <h1>: Headings such as <h1> provide structure to the content. They range from <h1> (most important) to <h6> (least important).
  • <p>: Paragraphs are used for text content within the body of the page.
  • </body>: This tag closes the <body> section.
  • </html>: This tag closes the <html> element and marks the end of the HTML document.

Example of Basic HTML Structure

Here's a simple example of a basic HTML structure:

            
                <!DOCTYPE html>
                <html>
                <head>
                    <title>My First Web Page</title>
                </head>
                <body>
                    <h1>Hello, World!</h1>
                    <p>This is a paragraph of text.</p>
                </body>
                </html>

HTML elements are usually nested within each other to create a structured and organized document. You can also include various other elements and attributes to add more functionality and style to your web page.

Now, let's explore more HTML elements and their usages in the subsequent pages. You can use HTML to create headings, paragraphs, links, lists, images, forms, and much more to build rich and interactive web content.
