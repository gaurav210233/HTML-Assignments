Page 2 - Links and Lists

HTML Links

HTML allows you to create hyperlinks to navigate between different web pages. Here's an example of an HTML link:

  • Visit Example.com

In this example, the <a> element is used to create a link. The href attribute specifies the URL to which the link points, and the target attribute is set to _blank to open the link in a new browser tab.

HTML Lists

HTML provides two main types of lists: ordered lists (<ol>) and unordered lists (<ul>). Lists are used to organize and present information in a structured manner.

Ordered List

An ordered list is a numbered list that represents a sequence of items. Here's an example:

  1. Item 1
  2. Item 2
  3. Item 3

Unordered List

An unordered list is a bulleted list that represents a collection of items. Here's an example:

  • Item A
  • Item B
  • Item C

You can use the <li> element to define individual list items within both ordered and unordered lists.

Now that you've learned about links and lists, you can create navigation menus, content organization, and much more in your HTML documents. Continue exploring different HTML elements and their uses in the upcoming pages.
