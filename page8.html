CSS Styling

This is an example of CSS styling applied to HTML elements. You can control the appearance of web pages using CSS (Cascading Style Sheets).

Styling Options

You can change various aspects of an element's style, such as:

  • Color
  • Text alignment
  • Font size
  • Line height
  • Background color
  • Margin and padding

This paragraph has a yellow background color applied using a CSS class.

By combining HTML and CSS, you have full control over the presentation and layout of your web pages.
